# RuoYi-Vue-UI 主题风格设置实现详解

## 一、现有主题风格

系统当前内置了三种主题风格：

| 主题标识      | 主题名称 | 菜单背景色 | 菜单文字颜色 | Logo标题颜色 |
| ------------- | -------- | ---------- | ------------ | ------------ |
| `theme-dark`  | 深色主题 | `#304156`  | `#bfcbd9`    | `#ffffff`    |
| `theme-light` | 浅色主题 | `#ffffff`  | `#303133`    | `#303133`    |
| `theme-blue`  | 蓝色主题 | `#2774f4`  | `#f0f0f0`    | `#ffffff`    |

---

## 二、主题设置的核心实现机制

### 1. 设置面板入口

**文件：** `src/layout/components/Settings/index.vue`

主题风格设置通过设置抽屉（Drawer）组件呈现，核心逻辑包括：

```vue
<!-- 主题风格选择区域（第30-64行） -->
<div class="setting-drawer-block-checbox">
  <!-- 深色主题 -->
  <div class="setting-drawer-block-checbox-item" @click="handleTheme('theme-dark')" title="theme-dark">
    <img src="@/assets/images/dark.svg" alt="dark" />
    <div v-if="sideTheme === 'theme-dark'" class="setting-drawer-block-checbox-selectIcon">...</div>
  </div>

  <!-- 浅色主题 -->
  <div class="setting-drawer-block-checbox-item" @click="handleTheme('theme-light')" title="theme-light">
    <img src="@/assets/images/light.svg" alt="light" />
    <div v-if="sideTheme === 'theme-light'" class="setting-drawer-block-checbox-selectIcon">...</div>
  </div>

  <!-- 蓝色主题 -->
  <div class="setting-drawer-block-checbox-item" @click="handleTheme('theme-blue')" title="theme-blue">
    <img src="@/assets/images/blue.svg" alt="blue" />
    <div v-if="sideTheme === 'theme-blue'" class="setting-drawer-block-checbox-selectIcon">...</div>
  </div>
</div>

<!-- 主题颜色选择器（第66-70行） -->
<div class="drawer-item">
  <span>主题颜色</span>
  <span class="comp-style">
    <el-color-picker v-model="theme" :predefine="predefineColors" @change="themeChange" />
  </span>
</div>
```

**核心方法：**

- `handleTheme(val)`: 切换主题风格（存储到 Pinia store 和 localStorage）
- `themeChange(val)`: 切换主题主色调（影响 Element Plus 组件颜色）

### 3. 主题处理工具

**文件：** `src/utils/theme.js`

```javascript
// 处理主题颜色（影响 Element Plus 组件）
export function handleThemeStyle(theme) {
  document.documentElement.style.setProperty('--el-color-primary', theme);
  // 生成 9 级浅色变体
  for (let i = 1; i <= 9; i++) {
    document.documentElement.style.setProperty(`--el-color-primary-light-${i}`, `${getLightColor(theme, i / 10)}`);
  }
  // 生成 9 级深色变体
  for (let i = 1; i <= 9; i++) {
    document.documentElement.style.setProperty(`--el-color-primary-dark-${i}`, `${getDarkColor(theme, i / 10)}`);
  }
}
```

### 4. 侧边栏颜色动态绑定

**文件：** `src/layout/components/Sidebar/index.vue`

```javascript
// 获取菜单背景色
const getMenuBackground = computed(() => {
  if (settingsStore.isDark) {
    return 'var(--sidebar-bg)';
  }
  if (sideTheme.value === 'theme-dark') {
    return variables.menuBg;
  } else if (sideTheme.value === 'theme-light') {
    return variables.menuLightBg;
  } else if (sideTheme.value === 'theme-blue') {
    return variables.menuBlueBackground;
  } else {
    return variables.menuBg;
  }
});

// 获取菜单文字颜色
const getMenuTextColor = computed(() => {
  if (settingsStore.isDark) {
    return 'var(--sidebar-text)';
  }
  if (sideTheme.value === 'theme-dark') {
    return variables.menuText;
  } else if (sideTheme.value === 'theme-light') {
    return variables.menuLightText;
  } else if (sideTheme.value === 'theme-blue') {
    return variables.menuBlueColor;
  } else {
    return variables.menuText;
  }
});
```

## 三、如何添加新的主题风格

以添加「绿色主题 (`theme-green`)」为例，完整步骤如下：

### 步骤 1：定义 SCSS 变量

**文件：** `src/assets/styles/variables.module.scss`

在变量定义区域添加：

```scss
// theme-green
$base-menu-green-color: #ffffff;
$base-menu-green-color-active: #ffffff;
$base-menu-green-background: #52c41a;
$base-logo-title-green-color: #ffffff;
```

在 `:export { ... }` 块中导出：

```scss
// theme-green
menugreencolor: $base-menu-green-color;
menugreencoloractive: $base-menu-green-color-active;
menugreenbackground: $base-menu-green-background;
logotitlegreencolor: $base-logo-title-green-color;
```

## 四、注意事项

1. **主题图标尺寸**：SVG 建议尺寸为 52px × 45px，保持与其他主题图标一致
2. **暗黑模式**：暗黑模式下所有主题都会使用统一的暗黑样式（定义在 `variables.module.scss` 的 `html.dark` 块中）
3. **localStorage**：主题设置会存储在 `layout-setting` key 下，修改主题标识符时要考虑向后兼容性
4. **组件库颜色**：`theme` 变量（主题颜色）控制 Element Plus 组件的主色调，与 `sideTheme`（侧边栏主题）是两个独立的概念
